# Thumbnail Platform

Premium demo-first web app for downloadable editable PSD thumbnail assets.

## Current build mode

This version is intentionally built without requiring external integrations.

It uses:
- mock auth
- mock subscription states
- mock assets
- mock favorites
- mock downloads
- placeholder billing flows

The goal is to build a high-quality working product shell first, then replace mock systems with real integrations later.

## Product goal

A premium members-only PSD asset library for YouTube creators and thumbnail designers.
