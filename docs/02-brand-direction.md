# Brand Direction

## Important Note

There is no final brand identity yet.

Create a temporary but high-quality brand direction that fits the product concept.

The temporary visual identity must feel intentional, premium, and trustworthy.
It must be easy to replace later with a custom logo and refined brand system.

You have creative freedom to propose and implement:
- temporary product naming directions
- a temporary wordmark approach
- a fitting color palette
- typography choices
- component styling direction
- layout rhythm and composition choices

As long as the result stays aligned with the documented concept and premium quality bar.

## Visual Personality

The product should feel:
- premium
- dark
- minimal
- modern
- polished
- restrained
- confident
- design-led
- high-end SaaS
- curated, not chaotic

## It should NOT feel like:
- a crypto casino landing page
- a startup template with loud gradients everywhere
- a generic marketplace
- a playful consumer app
- an overdecorated design portfolio
- a cheap downloadable assets site

## Design References

The feeling should be closer to:
- Linear
- Framer
- modern Apple restraint
- premium software dashboards
- high-end creative tools

## Creative Freedom Rule

Within the documented direction, make strong visual decisions.

Improve:
- layout quality
- visual hierarchy
- typography
- spacing
- navigation clarity
- card presentation
- preview presentation
- conversion clarity

If you see a better UI approach than a basic implementation, choose the better approach.

## UI Character

The interface should emphasize:
- excellent spacing
- clean typography
- strong hierarchy
- subtle borders
- sharp cards
- minimal visual noise
- large, attractive previews
- high content legibility
- elegant navigation
- calm, premium rhythm

## Color Direction

Use a dark premium base:
- deep neutral backgrounds
- slightly lifted panels
- subtle border contrast
- restrained accent color
- high readability
- soft highlight states

Do not overuse bright gradients or glowing effects.

## Typography Direction

Typography must feel:
- modern
- clean
- expensive
- highly legible
- calm and structured

Use a strong modern sans-serif system.

## Temporary Brand System

Until final branding exists:
- propose a few temporary product names
- choose one temporary direction
- use a wordmark-based placeholder brand in the UI
- do not depend on a custom logo illustration

## Product Presentation Rules

The previews of the assets are one of the main selling points.

The UI must present assets as:
- desirable
- premium
- useful
- professional

## Overall Requirement

If there is a conflict between "more effects" and "more premium restraint", choose premium restraint.
If there is a conflict between "safe generic UI" and "higher-quality premium composition", choose the higher-quality premium composition.
